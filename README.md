Clinical Research Vital Signs – R Script

Overview

This R script performs basic descriptive analysis of clinical vital signs data, focusing on temperature and pulse rate. It is intended for clinical research training and reporting.

⸻

Requirements

R packages:

lubridate
dplyr
tidyr


⸻

Data
	•	Input file: vs_raw.csv
	•	Key variables:
	•	IT.TEMP – Temperature
	•	PULSE – Pulse rate
	•	ACTUAL_ARMCD – Treatment arm

⸻

Functionality
	•	Reads raw clinical data
	•	Calculates summary statistics (mean, SD, median, min, max)
	•	Groups pulse rate summaries by treatment arm
	•	Generates formatted report outputs

⸻

Usage

Update the working directory path and run the script in R.
